# CareerFlow

CareerFlow is a full-stack job application tracker built with Next.js and PostgreSQL.

## Current Status

Stage 1 — Project Initialization

The project currently includes:

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui configuration
- Lucide React
- ESLint
- Prettier
- Initial project architecture
- Global loading, error, and not-found pages

## Requirements

- Node.js 20.9 or newer
- npm
- Git

## Development

```bash
npm install
npm run dev
```
